## On-page SEO standards

- Every page has a unique, descriptive `<title>` tag (50-60 characters, primary keyword front-loaded)
- Every page has a unique `<meta name="description">` (150-160 characters, includes CTA)
- One `<h1>` per page containing the primary keyword
- Logical heading hierarchy (`<h2>`-`<h6>`) with related keywords
- Alt text on all images (descriptive, not keyword-stuffed)
- Semantic HTML (`<article>`, `<nav>`, `<main>`, `<section>`)

## Technical SEO checklist

- Pages load in under 3 seconds (Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1)
- HTTPS everywhere
- XML sitemap submitted and up to date
- robots.txt allows crawling of important pages
- Canonical URLs on all pages to prevent duplicate content
- Structured data (JSON-LD) for articles, products, FAQs, breadcrumbs
- Mobile-first responsive design
- No orphan pages — every page reachable within 3 clicks from homepage
- Clean URL structure (descriptive slugs, no query parameters for canonical content)

## Content strategy framework

Topic cluster model:
1. **Pillar page** — comprehensive guide on a broad topic (2000+ words)
2. **Cluster pages** — focused articles on subtopics linking back to pillar
3. **Internal linking** — pillar links to clusters, clusters link to pillar and each other

Content quality signals:
- Comprehensive coverage (answer all related questions)
- Original research, data, or unique perspective
- Up-to-date information (refresh old content regularly)
- Clear structure (short paragraphs, lists, tables, headings)
- Author expertise signals (author bio, credentials)

## Competitor analysis template

For each competitor:
1. Top ranking pages — what keywords do they rank for that we don't?
2. Content gaps — what topics do they cover that we haven't?
3. Backlink profile — where are their links coming from?
4. Technical advantages — site speed, structured data, mobile UX
5. Opportunities — where are they weak and we can win?

## Reporting metrics

Monthly report should include:
- Organic sessions (trend vs previous month and YoY)
- Keyword rankings (top 10, top 30, new rankings)
- Top landing pages by organic traffic
- Click-through rate from SERPs
- Core Web Vitals scores
- New backlinks acquired
- Content published and optimised
- Action items for next month
