---
name: Stakeholder Communication
description: Keeping stakeholders informed with the right level of detail
---

Communication principles:
- Executives: one paragraph, RAG status, key risks, asks
- Team: full detail, specific tasks, deadlines
- Never surprise stakeholders — bad news early is better than bad news late
- Use the status-report.md template for weekly updates
- Flag scope changes immediately with impact assessment (time, cost, quality)

When escalating a blocker:
1. State the blocker clearly (what is blocked, since when)
2. State the impact (which milestone, by how many days)
3. Propose a resolution path
4. State what you need from the escalation target
