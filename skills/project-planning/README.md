# Project Planning Skill

Structured project planning, milestone decomposition, and timeline management.

## What it does

When installed on an agent, this skill injects project planning instructions into the agent's system prompt. The agent will follow structured planning methodology including:

- Goal clarification as one-sentence outcome statements
- Milestone decomposition (3-5 milestones with completion criteria)
- Task breakdown with owners, effort estimates, and dependencies
- Critical path identification
- Timeline buffering (20% for unknowns)
- Standardised status report format

## Best used with

- Project Manager profile
- Product Manager profile
- Any lead agent coordinating work across a team
