# Tailwind CSS Skill

Tailwind CSS best practices, design tokens, utility patterns, responsive design, dark mode, and v4 migration guide.

## What it does

Provides the agent with comprehensive Tailwind CSS knowledge including:

- Installation and setup for Vite and Next.js projects
- Tailwind v4 CSS-first configuration with `@theme` blocks
- Design token hierarchy (brand → semantic → component)
- Utility function patterns (`cn()`, CVA variants)
- Styling rules (semantic tokens, gap over space, size shorthand)
- Responsive design with mobile-first breakpoints
- Dark mode implementation with `next-themes`
- v3 → v4 migration reference
- Performance tips
- Build and run instructions

## Best used with

- Frontend Developer profile
- Full-Stack Developer profile
- Designer profile
- Any agent building UI with Tailwind CSS
