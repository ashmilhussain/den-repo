# shadcn/ui Skill

Component patterns, CLI usage, form composition, theming, and accessible UI development with shadcn/ui.

## What it does

Provides the agent with comprehensive shadcn/ui knowledge including:

- Project setup for Next.js and Vite
- CLI reference (init, add, search, docs, diff, dry-run)
- Full component catalog with install commands
- Component selection guide (which component for which need)
- Code patterns for forms (React Hook Form + Zod), dialogs, sheets, toasts, cards, data tables, charts
- Composition rules and accessibility requirements
- Styling rules (semantic tokens, no color overrides, cn() usage)
- Theming with CSS variables (light and dark)
- Component update workflow (diff before overwrite)
- Client vs server component guidance for Next.js
- Build and run instructions

## Best used with

- Frontend Developer profile
- Full-Stack Developer profile
- Designer profile
- Any agent building UI with React and shadcn/ui
