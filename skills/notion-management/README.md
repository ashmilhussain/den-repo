# Notion Management Skill

Best practices for organising Notion workspaces, designing databases, structuring pages, and managing content.

## What it does

Provides the agent with Notion-specific knowledge including when to use pages vs databases, workspace organisation patterns, database design templates (task tracker, content calendar, meeting notes, CRM), property type selection guide, page structure best practices, and Notion API usage tips.

## Best used with

- Project Manager profile (with Notion tool installed)
- Product Manager profile
- Any agent managing content or tasks in Notion
