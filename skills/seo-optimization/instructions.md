SEO best practices for content creation:

## On-page SEO
- Every page has a unique, descriptive `<title>` tag (50-60 characters)
- Every page has a unique `<meta name="description">` (150-160 characters)
- Use one `<h1>` per page that includes the primary keyword
- Use `<h2>`-`<h6>` for logical content hierarchy
- Include alt text on all images (descriptive, not keyword-stuffed)
- Use semantic HTML (`<article>`, `<nav>`, `<main>`, `<section>`)

## Content guidelines
- Write for humans first, search engines second
- Target one primary keyword per page
- Include the primary keyword in: title, H1, first paragraph, URL slug
- Use related keywords naturally throughout the content
- Aim for comprehensive coverage of the topic (long-form content ranks better)
- Internal linking: link to related pages within the site

## Technical SEO
- Ensure pages load in under 3 seconds
- Use HTTPS everywhere
- Create an XML sitemap
- Use canonical URLs to avoid duplicate content
- Implement structured data (JSON-LD) where appropriate
- Ensure mobile responsiveness

## Content structure
- Use short paragraphs (2-3 sentences)
- Include bulleted/numbered lists
- Add a table of contents for long articles
- Use descriptive anchor text for links (not "click here")
